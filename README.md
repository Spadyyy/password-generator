# 🔐 Memorable Password Generator (GUI)

A Python GUI app that generates strong yet memorable passwords using user-defined words.

## ✨ Features
- User-defined password length
- Custom number of words
- Clean and readable password structure
- Only one special character
- Smart placement of numbers and symbols
- Copy to clipboard feature

## 🧠 Example
Input:
Words → spandan, iem  
Output → Span42dan!Iem  

## ▶️ How to Run
python main.py

## 🛠️ Built With
- Python
- Tkinter
