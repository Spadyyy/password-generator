# 🔐 Memorable Password Generator (GUI)

Hey there! 👋  
This is a simple yet smart Python GUI app that generates **strong but easy-to-remember passwords** using your own words.

## 💡 Why this project?

Most password generators create something like: aK@92!xPz
Which is secure… but impossible to remember. So I built this tool to generate passwords like:
Tiger42!Sky , Span42dan!Iem


👉 Easy to remember  
👉 Still strong  
👉 Actually usable in real life  

## ✨ Features

- 🧠 **User-defined words** (you control the base)
- 🔢 Smart addition of numbers
- 🔐 Only **one special character** (clean & realistic)
- 🎯 Balanced & readable password structure
- 🖥️ Simple GUI using Tkinter
- 📋 One-click **Copy to Clipboard**
- 🚫 No spaces (valid everywhere)

## 🛠️ Tech Used

- Python 🐍  
- Tkinter (for GUI)
